About

Encoder plugins are components that allow adding custom encoder/decoder routines for Frhed. The encode/decode feature is available through Encode/ Decode-dialog in Frhed. There are two build-in encoders (XOR -1 and ROT13).

Frhed searches encoder plugins from Encoders-subfolder. Every DLL file in that folder is tried to load.

Installing and uninstalling

Just copy the encoder DLL file to Encoders-subfolder and it will be available next time the Encode/Decode-dialog is opened. Removing DLL file from the folder uninstalls the encoder. You can also change the filename extension to something other than .dll.

Creating own encoder

See the AddOne encoder project in EncoderPlugins\AddOne -folder. It implements very simple encoder which adds 1 to every byte.
